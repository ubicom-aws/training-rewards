# Training Rewards

Training Rewards is aimed at rewarding users for training they have completed.

## Quick Setup

```bash
git clone https://github.com/ubicom-aws/training-rewards.git
cd training-rewards

yarn
yarn dev
```

Access local instance from http://localhost:3000.

## Production build

```bash
git clone https://github.com/ubicom-aws/training-rewards.git
cd training-rewards

yarn
yarn build
yarn start
```
